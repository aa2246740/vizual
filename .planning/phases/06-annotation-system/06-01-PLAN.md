<objective>
Build the section renderer and multi-granularity annotation targeting, making DocView self-contained when used via json-render.

Purpose: When AI outputs `{ type: 'doc_view', sections: [...] }`, the Renderer instantiates DocView via the registry. The registry passes `{ props: { sections: [...] } }` -- NOT children. Currently, DocView only wraps `children` and ignores `sections`. This plan adds a section renderer that converts the sections array into rendered React content, and enables annotation on both text selections AND non-text elements (chart data points, KPI cards, table cells).

Output: DocView renders sections from AI JSON spec, text selection + annotation popup works, colored highlights render, clicking non-text elements triggers annotation.
</objective>

    The current AnnotationOverlay only handles text-based highlighting via react-highlight-words. For target-based annotations, we need a different approach: scan the DOM for `data-docview-target` elements and apply visual styles.

    After the existing `AnnotationHighlightWrapper` logic, add a `TargetHighlighter` component:

    ```typescript
    /**
     * Applies visual indicators to component targets (chart, kpi, table) that have annotations.
     * Uses MutationObserver to detect dynamically added targets.
     */
    function TargetHighlighter({
      annotations,
      containerRef,
    }: {
      annotations: Annotation[]
      containerRef: React.RefObject<HTMLElement | null>
    }) {
      useEffect(() => {
        const container = containerRef.current
        if (!container) return

        // Find all target annotations
        const targetAnns = annotations.filter(a => a.target)

        // Apply highlights to matching elements
        const applyHighlights = () => {
          // First, remove all existing target highlights
          container.querySelectorAll('[data-docview-annotated]').forEach(el => {
            const htmlEl = el as HTMLElement
            htmlEl.style.outline = ''
            htmlEl.style.outlineOffset = ''
            htmlEl.removeAttribute('data-docview-annotated')
          })

          // Apply highlights for each target annotation
          for (const ann of targetAnns) {
            if (!ann.target || ann.status === 'orphaned') continue
            const selector = `[data-section-index="${ann.target.sectionIndex}"][data-target-type="${ann.target.targetType}"]`
            const elements = container.querySelectorAll(selector)
            elements.forEach(el => {
              const htmlEl = el as HTMLElement
              htmlEl.style.outline = `2px solid ${ann.color}`
              htmlEl.style.outlineOffset = '2px'
              htmlEl.setAttribute('data-docview-annotated', ann.id)
            })
          }
        }

        applyHighlights()

        // Re-apply on DOM changes (sections may load async)
        const observer = new MutationObserver(applyHighlights)
        observer.observe(container, { childList: true, subtree: true })

        return () => observer.disconnect()
      }, [annotations, containerRef])

      return null
    }
    ```

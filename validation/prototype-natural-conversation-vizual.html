C — 完整接入包心智图 SDK / Runtime view

Agent

判断自然对话是否需要可视化升级；决定展示块出现位置；可声明 Catalog Gap metadata。

Vizual SDK

提供 Native Catalog、schema、validate、preview、render、action loop 和 gap report API。

Host

接入 SDK 的产品代码。负责展示消息 part、持久化 artifact、配置 gap sink。

Browser

真实渲染和真实交互验收位置。图表必须被人眼观察和点击。

自然对话可视化

  • 用户没有点名组件
  • Agent 自主选择 Native Catalog
  • 内联交互展示块进入消息流

显式创作请求

  • 用户明确要网页、游戏、HTML
  • 尊重创作路径
  • 不强制塞进 Vizual Native
presentVizualUi({
  input: nativeSurface,
  metadata: {
    catalogGaps: [{
      wanted: "interactive branch risk map",
      usedInstead: ["DataTable", "BarChart"]
    }]
  }
})

createVizualRuntime({
  catalogGapSink: {
    enabled: true,
    type: "file",
    path: ".vizual/gaps"
  }
})

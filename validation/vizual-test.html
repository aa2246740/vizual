Agent 对接指南

这个聊天页面暴露了一套 window API，任何能控制浏览器的 AI Agent 都能对接。核心流程：读用户消息 → 生成 JSON Spec → 渲染到气泡。

Bridge API（页面暴露的全局函数）

API说明返回
window._pendingMsg用户发送的最新消息（读完不会自动清，Agent 需置 null）string | null
window.createAiMsg()创建 AI 消息气泡（带流式光标），返回消息 IDstring (id)
window.streamText(id, text)往 AI 气泡追加文字void
window.finishText(id)结束流式输出（移除光标）void
window.renderVizInMsg(id, spec)渲染 Vizual JSON Spec 到气泡boolean
// 完整流程
const id = window.createAiMsg();         // 1. 创建气泡
window.streamText(id, '正在生成图表...');  // 2. 流式文字
window.finishText(id);                    // 3. 结束文字
window.renderVizInMsg(id, spec);          // 4. 渲染图表
window._pendingMsg = null;                // 5. 清除已读消息

Claude Code（Chrome DevTools MCP）

安装 chrome-devtools-mcp，Claude Code 自动获得 mcp__chrome-devtools__evaluate_script 工具：

// 监听用户输入
const msg = await mcp__chrome-devtools__evaluate_script({
  function: '() => window._pendingMsg'
});

// 回复
await mcp__chrome-devtools__evaluate_script({
  function: 'async () => {\n' +
    'const id = window.createAiMsg();\n' +
    'window.streamText(id, "这是回复文字");\n' +
    'window.finishText(id);\n' +
    'const resp = await fetch("/validation/my-spec.json");\n' +
    'const spec = await resp.json();\n' +
    'return window.renderVizInMsg(id, spec);\n' +
  '}'
});

Codex / Gemini / 通用方案（Playwright）

任何能跑 Node.js 的 Agent 都能用 Playwright 控制浏览器：

import { chromium } from 'playwright';

const browser = await chromium.launch({ headless: false });
const page = await browser.newPage();
await page.goto('http://localhost:8765/validation/vizual-test.html');

// 等用户输入 — 轮询 _pendingMsg
const msg = await page.evaluate(() => window._pendingMsg);

// 回复
await page.evaluate((spec) => {
  const id = window.createAiMsg();
  window.streamText(id, '这是回复文字');
  window.finishText(id);
  window.renderVizInMsg(id, spec);
}, mySpec);

纯 HTTP 轮询（无浏览器控制）

如果 Agent 完全无法控制浏览器，可以让页面自己轮询一个后端：

// 在页面 script 里加一个自动轮询
setInterval(async () => {
  if (window._pendingMsg) {
    const resp = await fetch('http://localhost:3000/chat', {
      method: 'POST',
      body: JSON.stringify({ message: window._pendingMsg }),
    });
    const { spec } = await resp.json();
    const id = window.createAiMsg();
    window.finishText(id);
    window.renderVizInMsg(id, spec);
    window._pendingMsg = null;
  }
}, 2000);

# Optional Catalog Gap Sink

Vizual should learn from natural conversation visualization cases where the Native Catalog lacks an expressive capability, but gap collection must not become a gate or a hidden side effect. A complete integration may configure a Catalog Gap Sink such as `.vizual/gaps/*.jsonl` or a host backend telemetry target; integrations without write capability simply skip collection while still rendering the current UI. This keeps the catalog improvement loop available without breaking agents or hosts that cannot write files.
